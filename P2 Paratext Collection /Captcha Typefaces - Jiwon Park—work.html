Captcha Typefaces


Typefaces
'The Designer is the Message'
workshop with Leonardo Sonnoli
Apr 2011





Have you taken a “Completely Automated Public Turing test to tell Computers and Humans Apart”? This might sound difficult and unfamiliar, however we sure all have taken it. This test asks us to prove we are human. This is also known as C.A.P.T.C.H.A.

A CAPTCHA is a program that generates and grade tests that humans can pass but computer programs cannot. Humans can read a distorted image that appears on the screen, but computer can’t. A CAPTCHA is commonly used to prevent unwanted automated internet bot from accessing websites. It is necessary for the web security. But do you like it? We all hate CAPTCHA. As we were asked to promote something we “hate”, I made a new way of using CAPTCHA.

CAPTCHA is readable by human but unreadable by computer.If we use this in a opposite way, we can fool the computer. I made two type of typefaces, Captcha_Facebook and Captcha_Google, which resembles actual CAPTCHA system that Google and Facebook uses. These CAPTCHA typefaces enable you to secure your own information safely. As it cannot be read by computer and OCR technology, you can prevent it from being detected by google search or other computer program.
